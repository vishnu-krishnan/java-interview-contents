### ITEM 7 ###

10 Golden Rules for Solving a Coding Question in an Interview 🔥

1 If we are dealing with
top/maximum/minimum/closest
'K' elements among 'N' elements, we will be using a Heap

2 If the given input is a sorted array or a list, we will either be using Binray Search or the Two Pointers strategy.

3 If we need to try all combinations (or permutations) of the input, we can either use Backtracking or Breadth First Search.

4 Most of the questions related to Trees or Graphs can be solved either through Breadth First Search or Depth First Search.

5 Every recursive solution can be converted to an iterative solution using a Stack.

6 For a problem involving arrays, if there exists a solution in O(n^2)time and O(1) space, there must exist two other solutions: 1) Using a HashMap or a Set for O(n) time and O(n) space, 2) Using sorting for O(n log n) time and O(1) space.

7 If a problem is asking for optimization (e.g., maximization or minimization), we will be using Dynamic Programming.

8 If we need to find some common substring among a set of strings, we will be using a HashMap or a Trie.

9 If we need to search/manipulate a bunch of strings, Trie will be the best data structure.

10 If the problem is related to a LinkedList and we can't use extra space, then use the Fast & Slow Pointer approach.

### ITEM 8 ###

In 2025, solving 1000+ questions on Leetcode ≠ getting a job.

Here’s How I Would Crack a Software Engineering Job in 2025 (Step-by-Step Roadmap for Indian Students)

► Phase 1: Build Your Foundations

1. Learn the Basics – Focus on Data Structures and Algorithms (DSA). Start with:
 - Arrays, Strings, Linked Lists, Stacks, Queues.
 - Platforms: GeeksforGeeks, HackerRank, and LeetCode (Easy/Medium).

2. Master Core CS Concepts – Parallelly work on:
 - Operating Systems – Learn memory management, scheduling, and deadlocks.
 - DBMS – Understand normalization, SQL queries, transactions, and indexing.
 - Computer Networks – Focus on TCP/IP, HTTP, DNS, and Load Balancing.
 - OOPs Concepts – Learn inheritance, polymorphism, and design patterns.

3. Resources:
 - CS50 by Harvard (Free on YouTube)
 - Neso Academy (DBMS, OS, and Networks on YouTube).
 - For DSA: Neetcode.io or CodeWithHarry (for Python/Java Basics).

► Phase 2: Problem-Solving Skills

1. Master Competitive Coding
 - Focus on Dynamic Programming, Greedy Algorithms, Graphs, and Binary Search.
 - Solve at least 150-200 problems on LeetCode and CodeForces.
 - Compete regularly in contests on CodeChef and AtCoder.

2. Advanced Topics
 - Learn Segment Trees, Tries, Bit Manipulation, and Disjoint Sets.
 - Practice SQL Queries for database handling scenarios.

3. Mock Tests and Time-Bound Practice
 - Set 30–40 minutes per problem. Simulate exam-like conditions.
 - Use InterviewBit and HackerRank to take mock tests.

► Phase 3: Build Projects & Resume

1. Focus on Real-World Projects
 - Build 2–3 projects showcasing Web Development (MERN/Java Spring Boot) or Backend APIs.
 - Add features like Authentication, Caching, Database Design, and Scalability.

2. Project Ideas
 - E-commerce Platform – Includes payment gateway, user login, and order tracking.
 - Social Media App – Features like posting, comments, and live updates using WebSockets.
 - System Design Demo – Scalable URL Shortener or File Storage App.

3. Resume Building
 - Highlight problem-solving ability, DSA progress, and project contributions.
 - Use tools like Novoresume for clean designs.

4. GitHub Profile & Portfolio
 - Keep all your projects documented on GitHub.
 - Create a personal website showcasing your work using React/Next.js.
